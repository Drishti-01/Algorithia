## 🤖 AI System Features

### What It Does
After you complete a question, the AI:
1. ✅ Analyzes your performance (time, attempts)
2. ✅ Calculates behavior score (0-100)
3. ✅ Determines strategy (support/probe/exploit)
4. ✅ Recommends next question
5. ✅ Shows beautiful popup with results

### Strategy Types
- 🟢 **Support** (Green): Easier questions for building confidence
- 🟠 **Probe** (Orange): Medium questions for testing boundaries
- 🔴 **Exploit** (Red): Harder questions to challenge weaknesses

### Interactive UI
- Performance score display
- Color-coded strategy indicator
- Next question recommendation card
- "Try Now" button for instant navigation
- "Continue Practicing" to dismiss

## 🎮 How to Use

### Step 1: Choose a Question
1. Go to Questions page
2. (Optional) Filter by category
3. Click "Start" on any question

### Step 2: Write Code
1. Use the Monaco editor on the right
2. Modify the starter code
3. Only the `solve()` function is editable

### Step 3: Run Simulation
1. Click "Run Simulation" button
2. Watch line-by-line execution
3. See visualization animate in real-time
4. Check variables and step log

### Step 4: Get AI Recommendation
1. Wait 1 second after completion
2. AI popup appears automatically
3. See your performance score
4. Get next question recommendation
5. Click "Try Now" or "Continue Practicing"

## 📝 Important Notes

### Code Constraints
- ❌ Don't use `!=` operator (use `==` with negation)
- ❌ Don't use multiple arrays (use individual variables)
- ✅ Use `for` loops instead of `while(condition != value)`

### Supported Operators

- ✅ `++`, `--`
- ✅ `+`, `-`, `*`, `/`

## 🐛 Troubleshooting

### If something doesn't work:

**Server not responding?**
```bash
# Stop and restart
npm run dev
```

**Visualization not showing?**
- Refresh the page (F5)
- Check browser console for errors

**AI popup not appearing?**
- Wait 1-2 seconds after simulation completes
- Check browser console for AI analysis logs

**Code not executing?**
- Check for syntax errors
- Ensure you're using supported operators
- Look at starter code for examples
